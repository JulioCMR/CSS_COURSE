Basic Layout (Box Model)

The CSS Box Model is type of CSS layout that uses boxes that wraps around every HTML element. It consists in: content, padding, border and margin.

Explanation of different parts:

  • Content: The content of the box (text or an image).

  • Padding: A clear internal area around the content. (Transparent)

  • Border:A border that goes around the content and the padding.

  • Margin:A clear external area outside the border. (Transparent)

Content

The main content of the element, it can be text or an image. There are two main properties: width and height. They will only change the characteristics of the content.

Exercises

  1. Change the width and height of an img element.

  2. Change the width and height of a p element.

    3. It doesn't change the size of the text, it will only change the size of the content text box.

Borders

A border that goes around the content and the padding. There are 3 main properties: size, style(with different styles, like solid, dashed, dotted and double) and color.

  1. Create a black, thin and solid border in an img element

  2. Create a green, thick and doubled border in an div element

    3. Some content for a better view
