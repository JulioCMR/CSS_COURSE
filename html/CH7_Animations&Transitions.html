The Transition property

Questions

  • What the transition property is used for?

    • It's a property used to style the transitional changes of properties. Example: Transitions on hovers.

  • What are the -webkit-, -moz- and -o- property prefixes are used for?

    • They are used to specify which browsers will use certain properties. Example: Webkit: IE and Safari, moz: Firefox, o: Opera. Used to work with older browsers versions.
