The Transition Property

Questions

  • What the transition property is used for?

    • It's a property used to style the transitional changes of properties. Example: Transitions on hovers.

  • What are the -webkit-, -moz- and -o- property prefixes are used for?

    • They are used to specify which browsers will use certain properties. Example: Webkit: IE and Safari, moz: Firefox, o: Opera. Used to work with older browsers versions.

Transformation Functions

Questions

  • what the transformation property is used for?

    • It's a property that uses functions to modify the "physical" characteristics of an element.

  • What are the functions used by the transformation property?

    • translate(), scale(), rotate(deg), skew(), matrix(), and their X and Y derivatives.

  • Mention what each of the transformation property functions does.

    • translate(): Modify the position of an element.
    scale(): Change the size of an element.
    rotate(): Change the rotation degree of an element.
    skew() Change the skewness of an element.
    matrix(): Shorthand property that summarize all the above properties.
