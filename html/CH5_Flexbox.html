Flexbox

  • What is Flexbox (Flexible Box)?

    • It's a form/type of display elements, based on the use of containers.

  • What are the main elements of the Flexbox?

    • A container and the child flex items inside it, all items will be arranged inside this container, sometime like a closet and all the elements you keep inside it.

Flex Direction & Wrap

Questions

  • What is the flex-direction property used for?

    • It's a property used to define the way in which the flex items are directed. The two main values are column and row.

  • What is the flex-wrap property used for?

    • It's a property used so that, in cases where there are multiple items, these are ordered in multiple lines, instead of compacting them in a single line.

Flex Item Order

Questions

  • What the order property is used for?

    • It's a property used for specify how the flex-items will be ordered in the container.

  • Why in most cases, is better to specify the order property in an inline style atrribute?

    • Because otherwise, you would have to create a different class on each item, and in each of them specify the order of those items.

Shrink, Grow and Basis

Questions

  • What the flex-basis property is used for?

    • It's used for set the base width of one or several flex-items.

  • What the flex-grow property is used for?

    • It's used for set how much a flex-item will grow relative to other items and the size of the viewport.

  • What the flex-shrink property is used for?

    • It's used for set how much a flex-item will shrink relative to other items and the size of the viewport.

  • Explain how the above properties help you to create a responsive web page.

    • I can look it in two ways:

    • In the browser window, each time you resize the window, the items on the web page will adapt and they will shrink or grow, depending of the size of the windows..
    • Looking on the different devices shapes viewpoint, the same adapt (shrink or grow) basis is used, this without creating device-specific code.
