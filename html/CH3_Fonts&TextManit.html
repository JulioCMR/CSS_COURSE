Fonts & Text Manipulation

Type of Units

There are two categories of units: Absolute and Relatives.

Absolute
Exact and standard measurements. Examples: cm, mm, pt(points), pc(picas), px(pixels).

Relatives
Units with a value related to the CSS values of others HTML elements. Examples: em, rem, vw, vh.
