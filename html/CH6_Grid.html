Grid

  • What is Grid?

    • Like Flexbox, it's a type of display used to order and display the elements of a web page.

  • What are the main differences between Grid and Flexbox?

    • A) The arrangement of the items is more manual.
    B) Not only focuses on the width like Flexbox does, is very focused on width and height.

Template Columns & Rows

Questions

  • What the grid-template-columns property is used for?

    • It's used to set the number and the size of the columns of the container.

  • What the grid-template-rows property is used for?

    • It's used to set the number and the size of the rows of the container.
